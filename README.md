# 手助网吧加盟结算系统 - 登录界面

## 项目简介
这是一个手助网吧加盟结算系统的登录界面原型，支持三种角色登录：
- 运营后台（内部管理员：OA登录）
- 代理（账号密码登录）
- 网吧主（账号密码登录）

## 功能特性
- 角色切换：三种角色登录方式
- OA登录：运营后台使用OA一键登录
- 账号密码登录：代理和网吧主使用
- 记住账号：网吧主可以选择记住账号
- 响应式设计：支持移动端访问

## 技术栈
- HTML5
- CSS3
- 原生JavaScript

## 使用方法
1. 直接用浏览器打开 `index.html`
2. 或使用本地服务器：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js
   npx http-server
   ```

## 演示账号

- 网吧主：owner001 / 123456
